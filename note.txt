1.创建版本库 git init

2.文件添加到仓库 git add  文件名称

3.文件提交 git commit -m "备注"

4.查看当前仓库状态  git status

5.查看修改内容 git diff

6.查看历史记录 git log

7.回到旧版本  git reset

8.删除文件  git  rm  "文件名称"

9.添加远程仓库  git remote add origin git@github.com:仓库名称.git

10.提交   git push origin master

11.删除远程连接  git remote rm "仓库名称"

12.克隆远程库  git clone "仓库地址"

13.创建分支并切换分支  git switch -c "分支名称"

14.查看当前分支  git branch 

15.合并分支  git merge "分支名称"

16.删除分支 git brand -d "分支名称"

17.切换分支 git switch "分支名称"

18.查看分支合并情况  git log  参数
